Intercom Messenger Webhook Server

Your server is running and ready to receive Intercom Canvas Kit webhooks.

Endpoints

  • Initialize Canvas: POST /initialize
  • Submit Canvas: POST /submit

How to use

  1. In your Intercom Developer Hub, set your webhook URLs to:
    • https://intercomwebhook.onrender.com/initialize
    • https://intercomwebhook.onrender.com/submit
    Replace your-server-url with your actual deployed server address.
  2. Use the Canvas Kit to build your Messenger app.
